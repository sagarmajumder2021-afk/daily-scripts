## 🚀 Quick Start

### Installation
```bash
# Clone repository
git clone https://github.com/sagarmajumder2021-afk/daily-scripts.git
cd daily-scripts

# Install Python dependencies
pip install -r requirements.txt

# Make bash scripts executable
chmod +x *.sh
```

### Usage
```bash
# Run Python script
python script_name.py

# Run Bash script
./script_name.sh

# Run JavaScript script
node script_name.js
```
